    JSON Generator tool - http://www.json-generator.com created by Vazha Omanashvili http://rainum.me/

    Modified

    Minified version from the site was parsed and modified to support more features by me

    Features Added

    1. Added support to generate a single Object. So {{repeat(10)}} is no longer required if generating only a single object.
    2. Can also generate a single value instead of object or Arrays of objects var template = '{{numeric(10,1000)}}';

    for testing and documentation: http://www.json-generator.com

    Generic Usage

        var jg = JsonGenerator.init({
        onGenerateComplete: function (r) {
            console.log(r);
        },
        onError: function (e) {
            console.log(e.name + ": " + e.message);
        }
    });

    jg.generateJson(template)
    console.log(jg.getJson());

    TEMPLATES

    Generate Array of objects

        var template = [
        '{{repeat(5)}}',
        {
            id: '{{index}}',
            guid: '{{guid}}',
            picture: 'http://placehold.it/32x32',
            age: '{{numeric(20,40)}}',
            name: '{{firstName}} {{lastName}}',
            gender: function (idx) {
                return this.bool() ? 'male' : 'female';
            },
            company: '{{company}}',
            phone: '{{phone}}',
            email: '{{email}}',
            address: '{{numeric(10000,40000)}}, {{city}}, {{street}}',
            about: '{{lorem(25)}}',
            registered: '{{date(YYYY-MM-ddThh:mm:ss Z)}}',
            tags: [
                '{{repeat(7)}}',
                '{{lorem(1)}}'
            ],
            friends: [
                '{{repeat(3)}}',
                {
                    id: '{{index}}',
                    name: '{{firstName}} {{lastName}}'
                }
            ]
        }];

    Generate a Value

        var template = '{{numeric(10, 1000)}}'; //Generates a number between 10 and 1000
    var template = '{{lorem(25)}}'; //Generates random Lorem Ipsum up to 25 characters
